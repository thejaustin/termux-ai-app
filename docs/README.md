# Termux AI App Documentation

This directory contains comprehensive documentation for the UX/UI improvements made to the Termux AI app.

## Documentation Files

- `UX-UI-Improvements.md` - Complete overview of all UX/UI improvements
- `Home.md` - Main documentation landing page
- `Mobile-Gestures-Guide.md` - Detailed guide on mobile gesture interactions
- `Accessibility-Features.md` - Information about accessibility enhancements
- `Claude-Code-Setup.md` - Claude integration features and usage
- `Architecture-Overview.md` - Technical architecture of the app
- `Wiki-Setup-Instructions.md` - Instructions for setting up the GitHub Wiki

## GitHub Wiki Setup

To transfer this documentation to the GitHub Wiki:

1. Visit the wiki page at: https://github.com/thejaustin/termux-ai-app/wiki
2. Follow the instructions in `Wiki-Setup-Instructions.md` to create wiki pages
3. Copy content from these files to the respective wiki pages

Alternatively, you can use the documentation files directly from this `/docs` directory, which is also a valid approach for project documentation.

## About the Improvements

The UX/UI improvements focus on making the Termux AI app more mobile-optimized with:
- Enhanced tab management
- Improved Claude status indicators
- Better accessibility features
- Mobile-specific gesture controls
- Quick settings panel
- Shake detection functionality
- Comprehensive help system

All changes maintain the core functionality while improving the user experience.
