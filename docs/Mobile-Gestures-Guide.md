# Mobile Gestures Guide

This guide describes all the mobile-specific gestures available in Termux AI to enhance your productivity on mobile devices.

## Available Gestures

### Swipe Gestures
- **Swipe Down**: Stop current Claude operation or dismiss status overlay
- **Swipe Left**: Navigate to the next terminal tab
- **Swipe Right**: Navigate to the previous terminal tab
- **Double Tap**: Show Claude message history or expand tab info
- **Triple Tap**: Activate voice input for Claude commands
- **Long Press Tab**: Show close button for the tab

### Touch Interactions
- **Long Press New Tab FAB**: Toggle bottom panel with quick actions
- **Long Press Settings Button**: Show quick settings panel
- **Tap Close Button**: Close the selected tab (appears after long press)

## Gesture Hints

The app provides visual hints for available gestures:
- A floating hint overlay appears when appropriate
- Contextual help is available in the Claude Help dialog
- Status indicators provide feedback during gesture operations

## Voice Input

The triple-tap gesture activates voice input functionality:
1. Triple tap anywhere on the terminal
2. Speak your command or question to Claude
3. The AI will process your voice input and respond appropriately

## Tab Management

Swipe gestures make tab management intuitive:
- Swipe left/right on the terminal area to switch between tabs
- Each tab maintains its own project context and Claude session
- Visual indicators show which tab is currently active
